#My Simple Portfolio Website

This is a simple and responsive portfolio website created using **HTML** and **CSS**.
The project contains a clean homepage and a contact form page to showcase basic frontend web development skills.

## Features

* Responsive homepage design
* Contact form section
* Simple and clean UI
* Beginner-friendly project structure
* Built only with HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Purpose of the Project

This project was created to practice and improve frontend development skills by building a basic personal portfolio website.

* Include projects section

* Connect contact form with backend/email service
